Sales Dashboard Project – End-to-End Demo

By Anandh Kumar Mallika Boobalan

Overview

This project demonstrates an end-to-end technical solution, combining data analysis, dashboarding, API integration, and web presentation — aligned with the responsibilities of a Data Analyst role in a public sector context.

  • HTML-based Web Embedding and Presentation: A static webpage was developed to host and explain the dashboard in a structured, reader-friendly format.
  • Power BI Dashboard: Created using mock sales data, it includes KPIs, revenue trends, product analysis, and regional insights with spatial visuals.
  • API Integration and Live Data Rendering: JavaScript is used to fetch product data from a public API and display it in real-time, simulating external data pull scenarios.

Together, these elements reflect a holistic understanding of data storytelling, stakeholder communication, and platform integration.
